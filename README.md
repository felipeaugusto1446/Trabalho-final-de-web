# Trabalho final de web: Gestor de Localizações

## Descrição
O **Minhas localizações maneiras** é um sistema para gestão inteligente de endereços. A plataforma permite que usuários autenticados salvem, editem e excluam seus locais favoritos, utilizando integração com a **BrasilAPI** para busca automática de dados via CEP.

## Tecnologias

### Front-end
* **React.js**: Framework usado para modularizar os componentes no frontend
* **Tailwind CSS**: Utilizado para estilização dos componentes e responsividade (não que tenha)
* **Vite**: Ferramenta de build

### Back-end
* **Python**: Linguagem principal de desenvolvimento
* **FastAPI**: Framework para gerenciamento das rotas no backend
* **Pydantic**: Validação de modelos de dados e segurança
